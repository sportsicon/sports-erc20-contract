## Commands:

1. `yarn` - to install deps
2. `yarn test` - to test contracts
3. `yarn deploy` - to deploy contracts (Rinkeby test network)
4. `npx hardhat flatten ./contracts/SportDao.sol > flat.sol` - to flat the contract
